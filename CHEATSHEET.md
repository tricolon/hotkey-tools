# AHK Cheat Sheet

## Hotstrings:
```
* → no ending character required to trigger the hotstring
? → hotstring will be triggered even when it is inside another word
c → case-sensitive
o → omit the ending character
x → execute single-line expression
```

## Hotkeys:
```
#    → win
^    → ctrl
<^   → left ctrl
>^   → right ctrl
!    → alt
<^>! → altgr
+    → shift
<!   → left shift
>!   → right shift
*    → wildcard
~    → native function will not be blocked
```
